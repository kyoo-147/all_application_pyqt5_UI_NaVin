## 前言
本项目是基于深度学习的人脸实时表情识别项目，方便部署运行，后续将继续更新。
表情识别部分采用TensorFlow+OpenCV实现，GUI使用pyqt5构建。  
目前可针对5种表情（Angry,Happy,Neutral,Sad,Surprise）进行识别。  
本项目已在Python3.7下测试通过。
